Build a full-stack web application called **GeoPolis: AI Geopolitical Simulator**.

## Overview

GeoPolis is a geopolitical strategy simulator where users take control of any country as its President, Prime Minister, Monarch, or Head of State. The player makes policy decisions, and an AI-powered simulation engine generates realistic economic, political, military, diplomatic, environmental, and societal consequences.

The experience should feel like a combination of a grand strategy game, a political simulator, and an AI-driven alternate history generator.

Use the Groq API for all narrative generation. The API key is already available in the `.env` file under the variable name `key`. Read it from environment variables and never hardcode credentials.

# Core Simulation State

Every nation must maintain:

```typescript
interface NationState {
  economy: {
    gdp: number;
    inflation: number;
    unemployment: number;
    debt: number;
    currencyStrength: number;
    foreignInvestment: number;
  };

  society: {
    happiness: number;
    literacy: number;
    healthcare: number;
    crimeRate: number;
    inequality: number;
  };

  politics: {
    approvalRating: number;
    corruption: number;
    stability: number;
    polarization: number;
  };

  military: {
    readiness: number;
    manpower: number;
    technology: number;
    nuclearCapability: number;
  };

  diplomacy: {
    influence: number;
    alliances: number;
    globalRelations: Record<string, number>;
  };

  environment: {
    pollution: number;
    energySecurity: number;
    climateRisk: number;
  };
}
```

# Turn-Based Engine

One turn equals one quarter.

Flow:

1. Show nation status
2. Generate events
3. Present decisions
4. Player chooses actions
5. Simulation updates
6. AI generates consequences
7. Advance to next turn
